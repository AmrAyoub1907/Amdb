# Amdb
Amdb is small Movie App which you can explore:<br/>
1-Top Rated<br/>
2-Most Popular<br/>
3-Up Coming<br/>
4-Now Showing<br/>
movies.<br/>
## Techneical Notes:
-written in **Kotlin**<br/>
-Used **RxJava**<br/>
-Used **Retrofit** beside Rx<br/>
-Used **Picasso** for caching and downloading images<br/>
-Used **Room** database<br/>
-**MVP** architecture is applied<br/>
